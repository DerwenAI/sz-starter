

## set up


```bash
python3 -m venv venv
source venv/bin/activate
python3 -m pip install -U pip wheel setuptools poetry
python3 -m pip install -r requirements.txt
```


## run

```bash
python3 pyvis-graph.py
```

Then load in your browser:

  * `./entity_graph1.html`
  * `./entity_graph2.html`
  * `./entity_graph3.html`
  * `./entity_graph4.html`
